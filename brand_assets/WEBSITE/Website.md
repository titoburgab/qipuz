## Design Principles

**Minimal interface** — Every element has a purpose. No decorative complexity.

**Typography-led** — Söhne headlines carry the brand personality. White space creates authority.

**Purposeful motion** — The Origin Node concept appears in transitions and loading states. Motion communicates the brand without interrupting the experience.

**Generous whitespace** — The brand commands space. Crowded layouts contradict the simplicity principle.

**Accessibility first** — WCAG AA minimum. Target AAA for primary content. See `DESIGN_SYSTEM/Accessibility.md`.

**Performance-first** — Core Web Vitals targets: LCP < 2.5s, CLS < 0.1, FID < 100ms.

## Tone

Professional. Confident. Human. Precise.

Never: hype, exaggeration, AI clichés, passive voice, vague claims.

See `BRAND_BOOK/Voice_Guide.md` and `WEBSITE/Copywriting.md` for detailed writing standards.

## Technical Standards

- Mobile-first responsive design
- WCAG AA accessibility (target AAA for primary content)
- Semantic HTML5
- Performance-optimized: images, fonts, JavaScript bundle
- SEO: meta, structured data, canonical URLs, sitemap
- Core Web Vitals targets met before launch
- No tracking without explicit user consent
