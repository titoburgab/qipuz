# QIPUZ Foundation Repository

**Version 2.0 — June 2026**

> *Engineering a future where intelligence is accessible, trustworthy, and seamlessly integrated into every organization.*

---

## What This Repository Is

This repository is the operating system of QIPUZ — an AI engineering company that designs intelligent systems, AI agents, and workflow automations.

It is the single source of truth for every decision made about the company: brand, design, engineering, product, AI strategy, marketing, and governance.

Every document is written to be actionable, not aspirational.

---

## Repository Structure

```
QIPUZ/
├── FOUNDATION/          # Company identity, values, mission, vision
├── BRAND_BOOK/          # Visual identity, logo, color, typography, motion
├── DESIGN_SYSTEM/       # Components, tokens, layout, accessibility
├── WEBSITE/             # Site architecture, UX, copy, SEO
├── AI/                  # Agent framework, governance, prompt architecture
├── MARKETING/           # Messaging, pitch, sales, content
├── PROMPTS/             # System prompts for AI-assisted workflows
├── TOKENS/              # Production-ready design token JSON files
├── SVG/                 # Vector logo and brand assets
└── ICONS/               # Icon system SVGs
```

---

## How to Use This Repository

- **New team member** → Start with `FOUNDATION/Manifesto.md`, then `FOUNDATION/Core_Values.md`
- **Designer** → Start with `BRAND_BOOK/Color_System.md` and `DESIGN_SYSTEM/Design_Principles.md`
- **Developer** → Start with `DESIGN_SYSTEM/` and `TOKENS/`
- **AI Engineer** → Start with `AI/AI_Strategy.md` and `AI/Agent_Framework.md`
- **Marketer** → Start with `MARKETING/Brand_Messaging.md`
- **Client** → Start with `FOUNDATION/Mission.md` and `FOUNDATION/Vision.md`
